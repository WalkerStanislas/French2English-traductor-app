# French2English-traductor-app
This is a streamlit application that load our translator model tained with European Parliament data using OpenNMT

## Fonctionnalities

- Translation of french sentences into english using OpenNMT
- Conviviale Interface build with Streamlit

## Installation

1. **Clone the repository :**

   ```bash
   git clone https://github.com/WalkerStanislas/French2English-traductor-app.git
   cd French2English-traductor-app
2. **Install dependancies**
   Be sure that you have python version 3.11, otherwise create a virtual env with that version
   Then run this command line:
   ```bash
   pip install -r requirements.txt
4. **Run the app**
   ```bash
   streamlit run app.py
5. **Model training**
   If you want to check the model training process, [visit my Kaggle account: ](https://www.kaggle.com/code/walkerstan/french2english-notebook).
6. **You can also test the translator app directly online** [French2English Translator](https://french2english-traductor-app-3dvgdhrkevboxpsnaazeyu.streamlit.app/)
